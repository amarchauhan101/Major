## 🚀 **Installation & Usage**

1. Load the extension in Chrome Developer Mode
2. Navigate to any website
3. Click the Cookie Guard icon
4. Hit "Scan Current Page" to see the magic!
5. Enjoy beautiful visualizations and easy cookie management
